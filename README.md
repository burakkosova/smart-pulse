# SmartPulse A.Ş. Stajyer Değerlendirme Teknik Mülakat Ödevi

Clone this repo:

`git clone https://github.com/burakkosova/smart-pulse.git`

Run `npm install` in order to install dependencies such as lodash

Then run `npm start`
